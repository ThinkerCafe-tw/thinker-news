1. SQL大升級！Google BigQuery整合AI函式，讓你在SQL裡直接玩轉LLM語意分析！

🔧 分類: AI工具與應用

Google 最新釋出的 BigQuery 託管人工智慧函式（Managed AI Functions），為資料分析師帶來劃時代的功能。這項更新提供三個可直接在 SQL 查詢中使用的 AI 函式：AI.IF、AI.CLASSIFY 和 AI.SCORE，讓使用者能無需編寫複雜的提示語或串接語言模型 API，就能進行語意分析任務。

例如，AI.CLASSIFY 可將文字自動分類，AI.SCORE 可用來對評論情緒評分，AI.IF 則可以用於條件式的語意運算。這些功能讓資料分析師在熟悉的 SQL 環境中，就能執行 NLP 任務，顯著降低了進入 LLM 應用的門檻。

💡 學習價值：
這是將 LLM 技術帶入資料分析日常流程的絕佳案例。若你熟悉 SQL，現在就能開始在查詢中整合 AI 功能，無需額外學習模型部署或 API 操作。

2. 開發者福音！VS Code重磅更新，新增Agent HQ一站式管理AI代理與Copilot，效率飆升！

🔧 分類: AI工具與應用

Microsoft 釋出 VS Code 的 10 月更新（v1.106），最亮眼的改版是新增「Agent HQ」介面，提供開發者集中管理 AI 工具的新方式。這讓使用者能在單一介面中，同時管理本地與雲端的 AI Agent、GitHub Copilot 擴充元件，並增強 AI 工具的核准與安全控管機制。

除了 AI 管理功能外，這次更新也優化了多項編輯體驗，提升了 Copilot 的整合性與穩定性。這些變動標誌著 AI 工具愈來愈深入地融入主流開發流程，未來 AI 將成為開發工作中不可或缺的一部分。

💡 學習價值：
若你使用 VS Code 進行 Python 或資料分析開發，掌握這些整合技巧將能極大提升開發效率，並熟悉 AI 開發環境的實戰工具鏈。

3. AI資安新星Runlayer重磅登場！獲Khosla創投$11M注資，八家獨角獸力挺，專攻AI代理安全！

🔧 分類: 資安趨勢快訊

知名創業家 Andrew Berman 推出新創公司 Runlayer，其核心業務是解決企業在使用 AI agent（代理）時的安全問題。Runlayer 已獲得來自 Khosla Ventures 和 Felicis Ventures 的 1100 萬美元種子資金，並聲稱獲得八家獨角獸企業支持，顯示 AI 安全市場的強烈需求。

隨著 AI agent 被廣泛用於自動化任務、客服、文件處理等領域，Runlayer 專注於確保這些代理在運作過程中不會產生資安漏洞或洩漏敏感資訊。這家公司正瞄準 AI 應用中的一大痛點：可信、安全的自動化。

💡 學習價值：
AI agent 安全是未來高需求領域。若你有資料安全或模型監控背景，Runlayer 的發展方向可為你提供職涯轉型的新契機。
