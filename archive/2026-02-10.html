1. OpenAI推出GPT-5.3-Codex：首個能「自我建立」的AI編碼模型，速度提升25%

OpenAI正式發布最新的AI程式撰寫模型GPT-5.3-Codex，標榜為公司首個「可自我建立」的模型。這表示它不只執行任務，還能部分自我設計運作流程，具備更高層次的自動化與自我改進能力。GPT-5.3-Codex專為所謂的「代理式編碼」（agentic coding）設計，能理解複雜的開發任務並自動分解成具體步驟。

在效能方面，OpenAI指出相較於前代版本，GPT-5.3-Codex在程式碼生成與重構任務上的速度提升了25%。這項技術的出現，意味著AI在軟體開發中的角色正快速由輔助者轉向主動參與者，甚至可能主導開發流程。

💡 學習價值：
初學者可藉此探索AI代理如何協助自動補全、除錯與重構程式碼，進而降低學習曲線。這也揭示可自我優化的AI模型將成為未來開發工具的主流。

2. AI讀遍數十年論文，科學家成功挖掘出25種全新高溫磁性材料

美國新罕布夏大學的研究團隊利用AI工具系統性分析數十年來的科學論文與材料資料庫，成功預測出25種全新高溫磁性材料。透過機器學習模型辨識材料結構與性質間的關聯，AI大幅加快了原本耗時的材料發現過程。

這些新材料能在更高溫下維持磁性，對於高性能電子設備與儲能技術發展具有關鍵影響。研究人員指出，AI的加入可說是徹底改變了材料科學的研究模式，從「試驗為主」轉向「資料驅動」。

💡 學習價值：
展現資料科學技術（如NLP與機器學習）如何應用於非傳統領域。對資料科學初學者來說，這是一個跨領域應用的典範，鼓勵將技能拓展至材料、醫學或能源等科學領域。

3. VS Code 1.109強化多代理人開發：引入工作階段管理與終端機沙箱

微軟發布Visual Studio Code 1.109版本，重點強化AI代理開發體驗。新版本加入了「代理工作階段管理」功能，讓開發者可在一個IDE中同時執行與管理多個AI代理，無論是本地、背景或雲端部署都能輕鬆切換。

此外，為防止AI代理操作終端機時造成安全風險，VS Code 1.109引入了「終端機沙箱」機制，可限制AI代理指令的權限與範圍，避免損害主機系統。

💡 學習價值：
對資料科學初學者而言，這代表你現在有一個更安全、集中且高效的環境來測試AI輔助開發流程。了解如何管理多個AI代理，將是進階開發技能的重要一環。

🛠 AI工具與應用焦點

AI社群平臺Moltbook配置大出包！攻擊者可冒名AI代理任意寫入資料

AI社交平臺Moltbook近日遭爆出嚴重資安漏洞。由於資料庫權限設定不當，導致API金鑰與用戶信箱等敏感資料外洩，更嚴重的是，任何人都能未經驗證即對資料庫進行讀寫。攻擊者甚至可冒用AI代理帳號發文或互動。

這起事件凸顯在開發AI應用時，資安設定的重要性不容忽視。即便平台功能創新，若基礎安全漏洞未處理，將可能造成重大損害。

💡 學習價值：
初學者在部署AI應用或API時，應理解「最小權限原則」與資料庫存取控管是基本功，不能只專注於功能實現。
