VS Code AI自動解Git衝突，Gmail AI幫排程，OpenAI聯手博通打造超級AI晶片

1. VS Code爆改再進化！「AI神隊友」自動解決Git衝突，新手也能輕鬆駕馭程式碼！

微軟在最近的 VS Code 9 月版本（v1.105）更新中，引入重大升級，其中最受矚目的新功能是加入了 AI 輔助解決 Git 合併衝突的功能。當開發者在使用 Git 進行程式碼版本控制時遇到合併衝突，AI 代理能自動生成解決方案，使用者可以直接在編輯器中審查與採納建議，減少處理合併問題所費的時間與精神力。這項功能對 Git 操作不熟的小白來說，無疑是一大福音。

除此之外，v1.105 還導入了多容器專案（MCP）伺服器市集預覽功能，支援直接透過 VS Code 安裝來自 GitHub 的各種 MCP 伺服器，簡化容器化開發環境的配置。

這些改進讓 VS Code 成為更強大的一站式開發環境，也反映 AI 正快速滲入軟體開發流程，協助開發者提升生產力與協作效率。

💡 學習價值： 初學資料科學者常會碰到版本控制與環境部署問題，此新功能有助於降低技術門檻，提高專案管理與協作效率，同時也幫助理解 AI 在實際開發工具中的應用場景。

2. Google Gemini大升級！Gmail也能幫你「秒排行程」，AI秘書功能直接開外掛！

Google 宣布將其 AI 助手 Gemini 更深地整合到 Gmail 中，推出全新的「Help Me Schedule」功能。當使用者在 Gmail 中撰寫或回覆電子郵件時，若內容提及排程需求（如「我們下週開會」），Gemini 就能根據上下文自動生成會議安排建議，包括時間、參與者與會議議程，並與 Google Calendar 和聯絡人資訊整合，智慧找出所有人都方便的時間。在點擊之間完成會議安排與邀請，使用者甚至不需離開 Gmail 界面。

此功能有望大幅簡化傳統繁瑣的會議協調過程，不僅提升工作效率，也讓大家更進一步體驗 LLM 在處理自然語言與行動指令上的效能。

💡 學習價值： 新手可以透過觀察 Gemini 如何理解文字上下文並做出邏輯推論，實際掌握 LLM 在「語境理解 + 自動執行」任務上的應用方式，這能提供許多未來 AI 應用開發的靈感與設計思路。

3. 重磅！OpenAI再出招：聯手博通打造「10GW超級AI晶片」！算力飆升、未來佈局大公開！

OpenAI 宣布與晶片大廠博通（Broadcom）合作，共同開發高達 10 吉瓦（GW）算力的客製化 AI 加速器。這象徵 OpenAI 開始從依賴通用 GPU（如 NVIDIA、AMD），進一步轉向針對其自家 AI 模型預訓練與推論所定制的硬體，期望兼具效能與成本的優化。

博通擁有在 ASIC（特定應用積體電路）設計方面的深厚技術，未來可能支援 OpenAI 打造自家專用的 AI 晶片，用以滿足龐大的模型訓練需求，同時降低對單一供應商依賴的風險。

這場合作標誌著 AI 龐大計算需求正在驅動硬體生態系統進化，客製化晶片成為推動新一代 AI 模型演進的關鍵。

💡 學習價值： 對資料科學家而言，這反映出效能已成為 AI 開發競爭力的核心之一，理解硬體架構的選擇對訓練效率的影響將逐漸成為必要基礎知識。未來也有更多不同規格、架構的 AI 加速器選擇值得評估與學習。

📊 產業趨勢與新聞

Fivetran、dbt Labs合併！打造資料處理超級組合拳

資料工程兩大廠商 Fivetran 與 dbt Labs 宣布全股票合併，目標整合 ELT（抽取、載入、轉換）資料處理流程。合併後將由現任 Fivetran 執行長擔任新公司 CEO，dbt Labs 開源工具 dbt Core 將持續維持開源狀態。

這項併購將除去許多資料流程維運障礙，提供一站式解決方案，也讓未來模型訓練的資料準備更加簡便與穩定。
